SPRINT-6 OBJECTIVE

• upload a dataset
• view job progress
• preview normalized data
• download generated artifacts

---

UI PAGES

Create a small UI with the following pages.

1. Home Page

Route:

GET /

Display:

• Upload dataset form
• Link to recent jobs

Upload form fields:

dataset_file (CSV or XLSX)

Submit action:

POST /prep/upload

After upload redirect to job status page.

---

2. Job Status Page

Route:

GET /jobs/{job_id}/view

Display:

• job status (NORMALIZING / COMPLETE / FAILED)
• dataset hash
• link to download artifacts when complete

If status is NORMALIZING, show refresh button.

---

3. Dataset Preview Page

Route:

GET /datasets/{dataset_id}/preview

Display:

• first 50 rows of normalized dataset
• column names
• inferred types from dataset_schema.json

Use pandas to read the normalized CSV and render a small HTML table.

---

4. Artifact Download Links

Display links for:

normalized_dataset.csv
normalization_report.md
dataset_schema.json
resolver_trace.json

These files already exist in storage; simply serve them.
