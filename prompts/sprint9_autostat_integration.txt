SPRINT-9 OBJECTIVE

Enable AutoStat Prep to send normalized datasets to AutoStat for statistical analysis.

AutoStat will run as a **separate service** accessed through an HTTP API.

---

CONFIGURATION

Add to app/config.py:

AUTOSTAT_API_URL
AUTOSTAT_API_TIMEOUT

Example:

AUTOSTAT_API_URL = "http://localhost:9000"

---

AUTOSTAT CLIENT

Create service:

app/services/autostat_client.py

Functions required:

send_dataset_for_analysis(dataset_id, user_id)

This function must:

1. Locate normalized dataset and schema.
2. Send them to AutoStat using requests.post().
3. Receive analysis results.
4. Save the results as analysis_report.json in the dataset directory.

---

API ROUTE

Add new endpoint:

POST /datasets/{dataset_id}/analyze

Steps:

1. Verify normalized dataset exists.
2. Call send_dataset_for_analysis().
3. Return analysis job status.

Display link to analysis report when available.

analysis_report.json
